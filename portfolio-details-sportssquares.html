Sports Squares

Project information

  • Category: Desktop Application
  • First Posted: 2/27/20
  • Project URL: https://github.com/MFBednarski3/SportSquares
  • Technologies Used: Java, JavaFX and Gradle

Based off the traditional home/office sports betting pool, this Java desktop application randomly generates numbers and names to complete squares ensuring a fair process and making certain there is no advantage to individuals.
